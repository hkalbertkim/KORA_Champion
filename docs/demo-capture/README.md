The capture package supports execution-path selectivity evidence and compute-weighted GPU demand reduction in benchmark workloads. It does not claim production savings, customer workload savings, real infrastructure savings, full provider/GPU live workload comparison, full 1M all-GPU measured execution, production representativeness, official validation, signed partner validation, broad workload superiority, or energy reduction proof.
