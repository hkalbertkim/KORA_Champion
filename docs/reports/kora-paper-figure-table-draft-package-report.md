## 7. Claim-Safe Summary

The paper asset package supports KORA as an execution-path selectivity system. It frames H100 as one route among deterministic, cache, CPU, provider, GPU, and fallback paths. It uses compute-weighted GPU demand reduction in benchmark workloads as the benchmark demand metric and bounded H100 routed subset measurement for measured routed GPU subset evidence.

The package does not imply deployed savings, customer workload outcomes, broad workload generality, official review status, complete million-request GPU-only measurement, or final publication readiness.
